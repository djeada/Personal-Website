Courses I developed

Kurs Podstaw Pythona [polish]

Detailed lessons on programming fundamentals using Python. We go from the basics of Python to concepts like concruency, networking and systems programming. There are also some examples of more modern Python features like templates, concepts and smart pointers.

Od C do Cpp [polish]

A course on C and C++ programming languages. It teaches the fundamentals of programming and how to use them in a practical way. We go from the basics of C to concepts like concruency, networking and systems programming. There are also some examples of more modern C++ features like templates, concepts and smart pointers.

Algorithms And Data Structures

Since I initially attended the course at my university in 2015, Algorithms and Data Structures have attracted my attention. I find it really beneficial to return to the fundamentals on a regular basis, especially while learning a new language. While learning, I usually find it helpful to compare my approach to that of others. Since this is my n'th iteration of the process, I wanted to share my code with others so that they, too, may benefit from it.
