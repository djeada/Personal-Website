Tetris

Tetrus is a game that was created by Alexey Pajitnov in 1984. The game is a tile-matching puzzle game where the player must rotate and place falling blocks to form horizontal lines without gaps. The player must fill the rows with blocks to clear them. The game ends when the blocks reach the top of the grid.

  • To move the block use the left and right arrow keys.
  • To rotate the block use the up arrow key.
  • To drop the block to the bottom of the grid use the space bar.
  • To restart the game use the escape key.
