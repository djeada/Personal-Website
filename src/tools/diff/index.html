Web-based Diff Tool

Enter two versions of text in the input boxes below and click "Compare" to see the differences.

Need Help?
  • Enter the Original Text in the first input area.
  • Enter the Modified Text in the second input area.
  • Click Compare to analyze the differences between the two texts.
  • Additions are shown with green highlighting and underline.
  • Deletions are shown with red highlighting and strikethrough.
  • Line numbers help you identify where changes occurred.
