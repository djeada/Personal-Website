Interaktywna konsola

Konsola Pythona to interaktywne środowisko, w którym można wpisywać i wykonywać pojedyncze polecenia. Aby otworzyć konsolę, należy wpisać polecenie python w wierszu poleceń. Wówczas zostanie wyświetlona informacja o wersji Pythona dostępnej w systemie.

<blockquote>
<blockquote>
<blockquote>
python
python 2.7.13 (default, Sep  2 2019, 20:42:59)

Z poziomu konsoli można wykonywać wszystkie dostępne polecenia Pythona. Po wpisaniu polecenia i naciśnięciu klawisza Enter, zostanie ono natychmiast wykonane i zwrócony ewentualny wynik (nie wszystkie komendy zwracają wynik). Polecenie help pozwala uzyskać informacje o dostępnych komendach i ich możliwych parametrach. Można również wywoływać funkcje z bibliotek, takich jak np. math.pi, która zwraca stałą pi.

<blockquote>
<blockquote>
<blockquote>
help
Type help() for interactive help, or help(object) for help about object.
3 + 4
7
print('Hello world')
Hello world
import math
math.pi
3.141592653589793
